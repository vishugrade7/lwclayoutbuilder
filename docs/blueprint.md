# **App Name**: Lightning Layout Composer

## Core Features:

- Row Configuration: Configure row settings such as multiple rows, horizontal and vertical alignment, and boundary pulling.
- Column Configuration: Configure column settings such as column type, padding, size, and device-specific settings.
- Add Column: Add new columns to the layout with configurable properties.
- Visual Layout Display: Display a visual representation of the configured layout in real-time.
- Code Generation: Generate LWC code based on the configured layout.
- AI-Powered Suggestions: The application uses AI tool that provides intelligent layout suggestions based on the desired design and content.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and modern feel.
- Background color: Light gray (#F5F5F5) for a clean and neutral background.
- Accent color: Blue Violet (#7952B3) to complement the primary color and add visual interest.
- Font pairing: 'Space Grotesk' for headlines and 'Inter' for body text, both sans-serif.
- Use a grid-based layout for easy arrangement of configuration options.
- Use modern, flat icons to represent different configuration options and tools.
- Subtle transitions and animations when adding or modifying columns for a smooth user experience.
